■概要■
  ファミコンソフトの開発支援ツール。
  1画面分の一枚画(256*240)から、キャラクター(CHR)とネームテーブル(NAM)を作成します。
  CHRを作成する際に、同じキャラはまとめられます。
  動作画面は screenshot*.png を参照してください。


■動作確認環境■
  OS  : Windows XP
  CPU : Intel Core2Duo
  RAM : 2GB
  ※必要性能ではありません。WindowsXPなら動作すると思います。


■インストール■
  ZIPファイルの内容をインストールしたいディレクトリへ展開してください。
  PicToChr.exeが実行ファイルなので、ショートカットを作ると便利です。

  ファイル一覧
    PicToChr.exe
    PicToChr_en.txt
    PicToChr_jp.txt
    sample1.bmp
    sample2.bmp
    screenshot*.png


■アンインストール■
  インストールしたファイルを削除するだけです。
  レジストリは使用していません。


■操作方法■
  (1)画像を開く
    一枚画(※画像形式は後述)をBMPファイルから開くか、別のソフトからコピーして貼り付けます。
    画像を開くと、自動的にCHRとNAMが内部で作成されます。
    画面右側のチェックボックスのON/OFFでNAMの表示を確認できます。

  (2)CHRを保存する

  (3)NAMを保存する


■対応画像形式■
  サイズ : 256*240以下
  色数   : 256色パレット
         ※パレットセットを考慮している必要あり

  パレットセットについてわからない場合、Webで調べるか、
  sample.bmp のパレット構成を見てください。
  それでもわからない場合、4色で画像を作成してください。

  画像はCHR化されるときに4色画像になります。
  パレット順に以下のように変換されます。
  0, 1, 2, 3, 0, 1, 2, 3, 0...
